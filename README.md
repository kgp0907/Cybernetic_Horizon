## Cybernetic_Horizon
Unity로 제작된 SCI-FI풍 액션 RPG 포트폴리오입니다.

미래, SF를 배경으로 한 다크소울같은 게임이 있으면 재밌을것 같다는 생각을 하여 제작하게 되었습니다. 

일반적인 액션 RPG처럼 플레이가 가능하며, 디자인패턴, 시네머신, 타임라인 , MemoryPool 등의 최적화가 되어 있습니다.

현재 Git 에는 프로젝트에서 구현된 소스파일만 있습니다.
